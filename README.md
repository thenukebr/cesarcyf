# Cesarcyf
BruteForce em cifras de César<br>
Dev by: VandalVNL<br>
Linguagem: Shell Script<br>

<b>Não há dependências para usar este programa.<b>

  ./cesarcyf "Olssv Mypluk" -19<br>
  $ [nn] > 	palavra<br>
  $ [19] >	Hello Friend<br>
</blockquote>
